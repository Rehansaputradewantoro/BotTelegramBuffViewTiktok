Ini adalah proyek sumber terbuka yang dibuat oleh Buffa, mohon jangan menggunakannya untuk tujuan penipuan apa pun!

Perintah yang termasuk dalam BOT:

view - jalankan tampilan sesuai link tiktok
daftar - melihat kunci yang ada (perintah admin)
getkey - dapatkan kunci untuk mengaktifkan paket
gen [gratis/vip] - buat kunci reguler atau vip (perintah admin)
rencana - melihat rencana saat ini
aktif - perintah aktivasi akun
clearkey - hapus kunci yang kedaluwarsa (botv2.py baru)

Modifikasi yang diperlukan untuk mengkonfigurasi bot:

Ubah dan tambahkan id admin, token bot di baris 12,13
Edit tautan yang dipersingkat untuk mendapatkan kunci di baris 266
Unduh perpustakaan yang diperlukan untuk bot: pip install -r setup.txt
Jika muncul error : No module name "..." maka gunakan perintah pip install...
Jika tampilan tidak muncul silahkan jalankan file view.py untuk mengecek error karena ini script utama untuk mem-buff tampilan

Jelaskan sedikit tentang perintah pembuatan RENCANA dan KUNCI
Paket GRATIS: Terbatas untuk menjalankan 500 tampilan/waktu dan perlu menunggu 60 detik setiap kali menjalankan perintah berikutnya
RENCANA VIP: Tidak ada batasan sama sekali

/gen free: menghasilkan 1 kunci GRATIS secara acak dengan tanggal kedaluwarsa 1 hari
/gen vip: menghasilkan 1 kunci VIP secara acak dengan masa berlaku 7 hari

Secara default, /getkey akan mendapatkan kunci GRATIS, jika Anda menginginkan kunci VIP, Anda perlu menghubungi ADMIN


Semoga Anda bersenang-senang menggunakan Bot ^_^
